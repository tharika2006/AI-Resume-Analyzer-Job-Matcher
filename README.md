# AI-Powered Resume Analyzer & Job Matcher

## Overview

This project is an NLP-based Resume Analyzer and Job Matcher developed using Python in Google Colab. It extracts information from resumes, compares it with a job description, calculates ATS and Match Scores, identifies missing skills, recommends courses, and generates a professional PDF report.

---

## Features

- Resume PDF Upload
- Resume Text Extraction
- Email & Phone Number Extraction
- Skill Extraction
- Job Description Analysis
- Resume–Job Matching
- Match Score Calculation
- ATS Score Calculation
- Missing Skill Detection
- Course Recommendations
- Skill Word Cloud
- Pie Chart Visualization
- PDF Report Generation

## How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Upload a resume PDF.
4. Enter a job description.
5. Run all cells.
6. Download the generated PDF report.
